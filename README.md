# Node starter bun

Start node project with bun

## Quick start 🏄

- [Node.js (v24 LTS)](https://nodejs.org/en/download/)
- [Git](https://git-scm.com/downloads)
- [Bun](https://bun.sh)

  > Clone the repository:

```
git clone https://github.com/salviega/node-starter-bun.git
```

> Install dependencies:

```
bun install
```

> Run the project:

```
bun run start:dev
```
